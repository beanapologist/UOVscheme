SilentVerify

Portable state certificates: bind a digest to chain roots (or other anchors), sign with UOV, verify with one public check — P(σ) = y.

Formally verified in Lean 4 — the cryptographic core is mechanized in UOVscheme/ (scheme correctness and related proofs; build with lake build in the repo root).

UOVscheme on GitHub — Lean proofs, Python statecert, WASM demo, optional chain API.

How it works

  1. Anchor → digest y
  2. Issue → signature σ
  3. Verify → P(σ) = y
