SilentVerify

Portable state certificates: bind a digest to chain roots (or other anchors), sign with UOV, verify with one public check — P(σ) = y.

Open research stack: UOVscheme on GitHub (Lean proofs + Python + EVM anchor registry).

How it works (plain language)

  1. Anchor — Turn chain state (or a cross-chain pair) into canonical bytes → a field digest y.
  2. Issue — A signer with a UOV secret key produces a short signature σ.
  3. Verify — Anyone checks P(σ) = y using only public data — no validator role.
